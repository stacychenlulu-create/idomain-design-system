# Stationery UI Kit — I Domain Industrial

Brand collateral that applies the identity to its most literal use — **stationery** — the very thing a corporate identity manual exists for.

## Run
Open `index.html` (static HTML + CSS, no build).

## Pieces
- **Business card** (90 × 54 mm) — front (white): horizontal logo lockup, copper accent rule, name + title, contact grid (Mobile / Tel / Email / Web). Back: solid digital-blue field with the reversed brushstroke mark and the `I DOMAIN INDUSTRIAL` tagline.
- **Letterhead** (A4) — logo top-left, corporate block top-right, serif document title, body copy, and a blue footer rule with `IDOMAIN.COM.TW`.
- **Layout spec panel** — restates the manual's page-5 rules (Noto Serif TC Semibold 15pt headings, Noto Sans TC Regular 9pt body, brand blue, copper accent, left-positioned logo).

## Source fidelity
Contact fields and the email pattern (`name@idomain.com.tw`) follow the manual; names are illustrative placeholders. Proportions are print-accurate. Swap in real personnel details to use.
